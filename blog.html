Important FAQ?

What is the Document Object Model (DOM)?

The document object model is a programming interface for web document that represent the structure of a webpage as a tree of object. we use dom to change content, style, structure of the webpage

How do you select an element from the DOM?

We can select an element form the dom using method like getElementByTagName() querySelector() getElementById() getElementByClassName()

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is attached to a parent element instead of multiple listeners on individual child elements.Event delegation method is useful because it makes your code simpler and more efficient, especially for lists or dynamic content.

How do you manipulate an element's attributes and styles using the DOM?

We can manipulate an element's attributes using the setAttribute() method. To change styles, use the style property, like element.style.color = 'red'
